== Description ==

A WordPress plugin for a highly customizable posts widget. Also works with custom post types.

You can edit the following options:

* Title
* Widget heading format (h2, h3, h4, strong)
* Post type
* Article content (excerpt, title or both)
* Excerpt length
* Number of posts to show
* Show "more" link
* Show hr
* Show link to all articles (and edit link text)

== Installation ==                                                                                                                                                
                                                                                                                                                                  
This section describes how to install the plugin and get it working.                                                                                              
                                                                                                                                                                  
1. Upload plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress                                                                                                    
1. Place the widget in your templates and edit the options
